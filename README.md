La voz de asesor de Bender, Futurama. Con algunas frases características pero manteniendo la coherencia del aviso original del juego. Si las frases se cortan a mitad, es debido a su larga duración, que pueden ser interrumpidas a mitad. Utilizadlo como queráis es un elemento cómico casual, nada serio. SOLO ESPAÑOL ESPAÑA.
